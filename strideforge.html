How It Works

THREE STEPS.
ZERO EXCUSES.

From idea to leaderboard in minutes. Build it, run it, prove it.

Forge a Challenge

Draw a route on the map, set checkpoints, define rules. Want runners to spell "HOPE" through downtown? Set it. Loop the park three times under 25 minutes? Done. Name it, tag it, publish it.
